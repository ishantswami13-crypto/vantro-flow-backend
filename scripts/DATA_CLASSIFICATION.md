# Data Classification Policy
## PUBLIC
- landing page content
- public marketing content
- public signed invoice/bill view with limited fields
## INTERNAL
- app metadata
- non-sensitive operational logs
- feature flags
## CONFIDENTIAL
- customer names, supplier names, invoices, purchases, sales, inventory, tasks, business settings
## RESTRICTED
- payment records, ledger/bank transactions, financial forecasts, cashflow, user auth/session data, API keys/secrets, webhooks, audit logs, admin/security logs
